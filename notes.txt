// PARTE 1
GraphQL es un lenguaje de consultas.

Diferencias entre Rest API y GRAPHQL -> Son cosas totalmente diferentes GRAPHQL es un lenguaje de consultas mientras que Rest API es una arquitectura
para recuperar recursos de una base de datos. De hecho GRAPHQL puede utilizar Rest API. No es una alternativa directa a Rest API con GraphQL puedes
perfectamente pedir datos a una Rest API,

Hay muchas formas de inicializar un proyecto con GraphQL hay muchas libererias, pero la mas famosa de todas es Apollo Server. Es una plataforma
que nos da no solo la forma de crear el servidor si no tambien un cliente, tambien tiene un cliente para React. 

Iniciamos el proyecto con npm init -y

Instalamos Apollo y GraphQl -> npm install apollo-server graphql
